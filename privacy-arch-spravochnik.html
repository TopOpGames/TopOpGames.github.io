Arch-Reference Арх-Справочник

Последнее обновление: 16 апреля 2026 года

Настоящая Политика конфиденциальности описывает, как приложение «Арх-Справочник» (далее — «Приложение») обращается с информацией пользователей.

1. Сбор информации

Приложение не собирает, не хранит и не передаёт персональные данные пользователей. Мы не запрашиваем имя, email, номер телефона, местоположение или любые другие идентифицирующие сведения.

2. Локальное хранение

Приложение сохраняет на устройстве пользователя следующую информацию:

  • Настройки интерфейса
  • Ориентация экрана
  • Счётчики показов рекламы

Эти данные хранятся только локально на устройстве и никуда не передаются. При удалении приложения они полностью удаляются.

3. Сторонние сервисы

Приложение использует Yandex Mobile Ads SDK для показа рекламы. Рекламная сеть может собирать:

  • Рекламный идентификатор устройства (GAID/OAID)
  • IP-адрес
  • Модель устройства, язык, часовой пояс
  • Тип сети (Wi-Fi / мобильная)

Эти данные используются исключительно для показа релевантной рекламы. Политика конфиденциальности Яндекса.

Яндекс — российская компания, и все данные, собираемые Yandex Mobile Ads SDK, обрабатываются и хранятся на серверах Яндекса, расположенных на территории Российской Федерации, в соответствии с Федеральным законом № 152-ФЗ «О персональных данных».

4. Разрешения

Приложение запрашивает разрешение на доступ к интернету и рекламный идентификатор. Никаких других разрешений (камера, микрофон, местоположение, контакты) не требуется.

5. Дети

Приложение подходит для всех возрастов. Мы не собираем никаких данных о детях или взрослых.

6. Безопасность

Все сетевые запросы осуществляются по протоколу HTTPS.

7. Права пользователя

Пользователь может отключить персонализацию рекламы в настройках устройства или удалить все локальные данные путём удаления приложения.

8. Изменения

Данная политика может быть обновлена. При существенных изменениях информация будет обновлена на этой странице с указанием новой даты.

9. Контакты

This Privacy Policy describes how the "Arch-Reference" application (the "App") handles user information.

1. Information Collection

The App does not collect, store, or transmit any personal user data. We do not request your name, email, phone number, location, or any other identifying information.

2. Local Storage

The App saves the following information on the user's device:

  • Interface settings
  • Screen orientation
  • Ad display counters

This data is stored locally on the device only and is not transmitted anywhere. It is completely deleted when the App is uninstalled.

3. Third-Party Services

The App uses Yandex Mobile Ads SDK to display advertising. The ad network may collect:

  • Device Advertising ID (GAID/OAID)
  • IP address
  • Device model, language, timezone
  • Network type (Wi-Fi / mobile)

This data is used solely for displaying relevant advertisements. Yandex Privacy Policy.

Yandex is a Russian company, and all data collected by the Yandex Mobile Ads SDK is processed and stored on Yandex servers located in the Russian Federation, in accordance with Federal Law No. 152-FZ "On Personal Data".

4. Permissions

The App requests internet access and advertising identifier permissions. No other permissions (camera, microphone, location, contacts) are required.

5. Children

The App is suitable for all ages. We do not collect any data from children or adults.

6. Security

All network requests are made via HTTPS protocol.

7. User Rights

Users can disable ad personalization in device settings or delete all local data by uninstalling the App.

8. Changes

This policy may be updated. In case of significant changes, information will be updated on this page with a new date.

9. Contact

For privacy-related questions: TopOpGames@yandex.com
